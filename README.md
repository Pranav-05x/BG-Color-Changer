# BG Color Changer

## About
A simple web application that allows users to change the background 
color of the page by clicking colored buttons or entering a custom color.

## Features
- Click on colored buttons to change background instantly
- Enter any custom color name in the input field
- Default button to reset background to white

## Tech Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Open index.html in your browser
